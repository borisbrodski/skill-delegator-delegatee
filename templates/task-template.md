```markdown

<Clear, concise description of what needs to be accomplished>

1. <Requirement 1>
2. <Requirement 2>
3. <Requirement 3>

## Constraints

- <Constraint 1: e.g., "Must use existing OAuth2 library">
- <Constraint 2: e.g., "No breaking changes to API">

## Deliverables

- [ ] <Deliverable 1>
- [ ] <Deliverable 2>
- [ ] <Documentation>
- [ ] <Tests>

## Notes

<Additional context, links to related issues, or special instructions>
```
